# EcoTech CP ♻️💻

**Projeto 1 - Programação Web Front-End**
**Universidade Tecnológica Federal do Paraná (UTFPR) - Campus Cornélio Procópio**

## 🎯 Sobre o Projeto
O **EcoTech CP** é uma plataforma desenvolvida para facilitar o descarte consciente e a reciclagem de lixo eletrônico (e-waste) na região de Cornélio Procópio. O sistema conecta pessoas físicas e jurídicas que possuem equipamentos obsoletos com pontos de coleta e empresas de reciclagem da região.

Este projeto foi construído utilizando exclusivamente **HTML5** e **CSS3** puros (sem a utilização de frameworks), focando na correta aplicação de tags semânticas, seletores CSS e validação junto à W3C.

## 🌍 ODS Atendidos
A plataforma foi desenhada para contribuir diretamente com a agenda global, focando em:
* **ODS 9:** Indústria, Inovação e Infraestrutura.
* **ODS 12:** Consumo e Produção Responsáveis.
